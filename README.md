This app is a new version of an older app developed previously.
It consists in a chatting app for users, only for 1 to 1 conversations.
Development is following hexagonal architecture and also the core(domain and application) had been created using TDD(Test-Driven Development).
Framework used for testing is JUnit5 and mockito.
Documentation is done with the tool Javadoc.
Actually, it is only usable by CLI. It is planed to develop a UI using Swing.
